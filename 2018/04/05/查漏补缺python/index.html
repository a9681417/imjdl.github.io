python的驻留(interning)机制

其实在python中，整数和字符串都采用了一种驻留机制。

对于整数而言 范围是-5 ~ 256

对于字符串则是仅包含0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ_abcdefghijklmnopqrstuvwxyz的字符串，注意是仅包含，上面的图片中加了个空格，空格不再驻留机制中，所以不是同一对象。

赋值

在c语言中，赋值时就在内存中开辟了一块空间，然后把值放进去。

运行结果:

然而在python中却不是这样

可以发现a与b指向了同一块内存，这也是与上面所说的驻留机制有关。python 中一个变量可以说是一个标签，就好像粘上去的,当一个对象没有任何标签引用它的时候，就会被回收。所以，python中的变量只是一个标识符，赋值就是让标志符指向内存空间，因此python中的变量没有类型，类型是属于对象的，这也是python中的变量可以被任何类型赋值的原因。

浅拷贝

其实以前一直误以为，浅拷贝就是赋值，后来了解到并不一样。

注意:浅拷贝和深拷贝的不同仅仅是对组合对象来说，所谓的组合对象就是包含了其它对象的对象，如列表，类实例。而对于数字、字符串以及其它“原子”类型，没有拷贝一说，产生的都是原对象的引用。

所谓的浅拷贝就是创建一个新的对象，内容是元对象中元素的引用，新建一个组合对象，不拷贝子对象

常见的浅拷贝:切片操作、工厂函数、对象的copy()方法、copy模块中的copy函数

深拷贝

深拷贝就是创建一个新的对象，并递归拷贝子对象，深拷贝出来的对象与原对象没有关联。
