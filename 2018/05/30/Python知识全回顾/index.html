0x01 Python 中的基本数据类型，基本语法和语言构造

基本数据类型

  Python 中的数据类型分为六个: Number(数字)、String(字符串)、List(列表)、dict(字典)、tuple(元组)、set(集合)，其中 List和dict为可变对象，其他的都为不可变对象。在Python中Number中有int(整数)、float(浮点数)和complex(复数)。

数据类型间的转换
函数 描述
int(x, base) 把一些非int类型的数值转化为整数，如:int(0xf)会输出15，当一些字符串的时候就可以去指定base参数，base默认为10，意为转化为10进制，当然你也可以填0或2-36之间的数，如: int(‘0xf’, base=16) 输出15
float(x) 将字符串或数字转换为浮点类型，如: float(1) 输出 1.0 float(‘1’) 输出 1.0
bin(x) 将一个整数转换为二进制，以字符串输出。如: bin(4) 输出 ‘0b100’
complex(real [,imag]) 返回一个复数
str(x) 将对象x转化为一个字符串，创建出一个新的字符串对象并返回。
repr(x) 将对象 x 转换为表达式字符串，可以通过eval再次转化为对象
eval(x) 用来计算在字符串中的有效Python表达式,并返回一个对象
tuple(x) 将序列 x 转换为一个元组
list(x) 将序列 x 转换为一个列表
set(x) 转换为可变集合(可以去重)，集合三大特性:确定性、互异性和无序性
dict(x) 创建一个字典。x 必须是一个序列 (key,value)元组。
frozenset(x) 返回一个冻结的集合，冻结后集合不能再添加或删除任何元素。
chr(x) 将一个整数转换为一个字符 ascil 码
ord(x) 将一个字符转换为它的整数值,与chr相逆
hex(x) 将一个整数转换为一个十六进制字符串
oct(x) 将一个整数转换为一个八进制字符串
